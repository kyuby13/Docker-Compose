3. Edit sream yang di parent

- masuk ke docker parent

docker exec -it netdata-parent bash

- edit file

nano /etc/netdata/stream.conf

- tambahkan file

[key-child-b]
    enabled = yes
    default history = 3600
    default memory mode = save

- restar service 

docker restart netdata-parent
